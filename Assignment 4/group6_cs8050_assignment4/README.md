## Installation

To get started with DAA2 Assignment Implementations, follow these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/vaniseth/-DESIGN-AND-ANALYSIS-OF-ALGORITHMS-II.git
cd -DESIGN-AND-ANALYSIS-OF-ALGORITHMS-II
```

### Run the code:

```bash
cd [path-to-project]/group6_cs8050_assignment4
java -cp out/production/group6_cs8050_assignment4 mst.MSTAssignment
```
