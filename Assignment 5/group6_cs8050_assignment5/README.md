# CS 8050 Assignment 5: Machine Learning Algorithms Analysis
## Running the Code

### Prerequisites
```bash
pip install tensorflow scikit-learn matplotlib seaborn psutil numpy
```
or 
```bash
pip install -r requirements.txt
```

### Random Forest
```bash
python random_forest_mnist.py
```

### CNN
```bash
python cnn_mnist.py
```

### Visualizations
```bash
python generate_visualizations.py
```
Generates all figures used in the report.
